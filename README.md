# Jogo qualquer em pygame

## 1. Baixe esse reposit√≥rio
Simplesmente baixe o ZIP e descompacte

## 2. Crie um ambiente virtual no seu pc
No prompt de comando/terminal, dentro da pasta do jogo, rode:
```bash
python -m venv env
```

ou

```bash
python3 -m venv env
```

## 3. Ativar o ambiente virtual
No Windows PowerShell rode:
```bash
env\Scripts\Activate.ps1
```

No Windows Prompt de Comando rode:
```bash
env\Scripts\activate.bat
```

No Linux/MacOS rode:
```bash
source env/bin/activate
```

## 4. Instale os requerimentos
No prompt de comando/terminal rode:
```bash
pip install -r requirements.txt
```

## 5. Inicie o jogo
No prompt de comando/terminal, dentro da pasta do jogo, rode:
```bash
python src/main.py
```

ou

```bash
python3 src/main.py
```

## 6. Divirta-se
